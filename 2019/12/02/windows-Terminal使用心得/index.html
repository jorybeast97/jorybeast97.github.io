Windows Terminal

Windows Terminal是微软推出的一款新的命令行工具，对于windows用户来说，苦CMD久矣。最近换了windows terminal，感觉不错，可以使用一下。

首先来看看官方的介绍。

We are beyond excited to announce Windows Terminal! Windows Terminal is a new, modern, fast, efficient, powerful, and productive terminal application for users of command-line tools and shells like Command Prompt, PowerShell, and WSL.

多窗口支持

这次Terminal增加了多窗口的支持，相比较以前所使用的CMD来说简洁了很多，切换也变得更加方便，同时其支持多个命令行应用程序，nice！

更加好看的界面

terminal支持了很多字体界面的设置，使用对于程序员很友好的json格式，虽然我自己很喜欢原来自带的界面，但是如果你想要DIY一个自己的Terminal，这是个很好的选择。

关于如何设置界面的问题，网上有很多文章，这里不再赘述。

开源

同时，Terminal是开源的，可以在issue能看到很多对于其bug的反馈和改善。
